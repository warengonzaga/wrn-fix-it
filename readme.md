## ⚡ Features

- Tools
  - DNS Server Changer
    - Default / DHCP
    - Google DNS
    - Cloudflare DNS
    - Freenom DNS
    - Comodo DNS (New)
    - Quad9 DNS (New)
    - Verisign DNS (New)
    - OpenDNS (New)
    - AdGuard (New)
    - Custom (New)
  - More tools
- Modules (coming soon)
  - More modules

## 💻 Supported Systems

- Windows 10 (Batch version)
- Windows 11 (Batch version)

## 🕹️ Usage

Download the repository and look for **``"wrn-fix-it-vX.X.X.bat"``** file and run ``*.bat`` file as administrator. All you have to do is to follow the on-screen instructions.
